### 점과 선의 수에 따른 부호의 가짓수

1 => 2  
2 => 2 * 2 = 4  
3 => 2 * 2 * 2 = 8  
4 => 2 * 2 * 2 * 2 = 16

2 + 4 + 8 + 16 = 30

4개의 점과 선으로 30가지의 문자를 표현할 수 있습니다.

이진 개체나 모스부호와 같은 이진 부호는 항상 2의 거듭제곱수로 그 결과를 표현할 수 있습니다.
